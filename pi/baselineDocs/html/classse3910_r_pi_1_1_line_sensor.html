◆ taskMethod()

void se3910RPi::LineSensor::taskMethod ( )
virtual

This is the task method. The task method will be invoked periodically every taskPeriod units of time.

1.0 See if there is a command in the queue for us. If there is, process it.

1.0.1 There is an event. Dequeue the event from the queue.

1.0.2 If the event is to stop line sensing, set currentActive to false.

1.0.3 If the command is to start line sensing, set currentActive to true.

1.0.4 If the command is to stop line following, set line following enabled to be false.

1.0.5 If the command is to enable line following, set line following enabled to be true.

2.0 If we are to be doing line sensing.

2.0.1 Read in all three sensors into local variables.

2.0.2 If all three sensors are high, send a stop command. This should happen no matter whether line following is enabled or not.

2.0.3 Otherwise, if line following is enabled,

2.0.3.1 Based upon the in values, send commands to the motors to steer the robot. If black in the middle and white on the sides, make the robot go straight..

2.0.3.2 If black on the left and white on the right, make the robot go left.

2.0.3.3 If black on the right and white on the left, make the robot go right.

2.0.3.4 If black on the right and left and white in the middle, make the robot stop.

Otherwise, do nothing and just leave the robot doing what it did before.
