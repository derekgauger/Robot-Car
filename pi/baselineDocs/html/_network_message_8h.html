DESCRIPTION

This file defines a network message. A network message is sent over the socket to control the robot. Within this network message there are three parts. The first 32 bit integer defines the destination for the message. The second 32 bits defines the specific message. The third 32 bits is a checksum which verifies that the message is correct.
