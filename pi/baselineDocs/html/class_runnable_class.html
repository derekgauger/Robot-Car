Detailed Description

This is the runnable class, which mimics the runnable interface from Java. It is a virtual class which should not directly be instantiated.

◆ startChildRunnables()

void RunnableClass::startChildRunnables ( )
virtual

This method will start up any runnable objects which are contained within a class that implements the RUnnable interface. If there are no other objects that are runnable, there is no need to override this method. However, if a child class contains an instance of a runnable object, then this method must be overridden in the derived class.

This default implementation does nothing.
